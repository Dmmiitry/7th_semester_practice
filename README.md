# 7th_semester_practice

Пример реализации серверной части (Backend) для кейс-задачи № 3. Код написан на языке Python с использованием асинхронного фреймворка FastAPI. 

Данный микросервис реализует логику создания запроса на выплату, проверку баланса компании и имитацию взаимодействия с банковским шлюзом.

Структура проекта:
7th_semester_practice/

├── main.py              # Точка входа приложения, роутеры

├── models.py            # Pydantic-схемы данных (Request/Response)

├── services.py          # Бизнес-логика: расчеты, валидация

└── db.py                # Имитация базы данных (In-memory storage)

Файл main.py (Точка входа FastAPI)
Определение HTTP-эндпоинтов и обработка исключений.

Файл models.py (Схемы данных)
Используется библиотека Pydantic для описания структуры входящих запросов и исходящих ответов API.

Файл services.py (Бизнес-логика)
Здесь происходит расчет комиссий, проверка лимитов и логика смены статусов.

Файл db.py (Имитация хранилища)
В реальном проекте здесь была бы интеграция с PostgreSQL или ClickHouse. Для демонстрации используется словарь в памяти.
